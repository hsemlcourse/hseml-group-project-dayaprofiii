## Описание задачи

**Задача классификация:** Необходимо предсказать популярен ли музыкальный трека по его аудио-характеристикам

**Датасет:** Spotify Tracks Dataset (https://www.kaggle.com/datasets/yashdev01/spotify-tracks-dataset)

**Целевая метрика:** F1

## Структура репозитория

Опишите структуру проекта, сохранив при этом верхнеуровневые папки. Можно добавить новые при необходимости.

```
.
├── data
│   ├── processed               # Очищенные и обработанные данные
│   └── raw                     # Исходные файлы
├── models                      # Сохранённые модели
├── notebooks
│   ├── 01_eda.ipynb            # EDA
│   ├── 02_baseline.ipynb       # Baseline-модель
│   └── 03_experiments.ipynb    # Эксперименты и ablation study
├── presentation                # Презентация для защиты
├── report
│   ├── images                  # Изображения для отчёта
│   └── report.md               # Финальный отчёт
├── src
│   ├── preprocessing.py        # Предобработка данных
│   └── modeling.py             # Обучение и оценка моделей
├── tests
│   └── test.py                 # Тесты пайплайна
├── requirements.txt
├── README.md
├── .gitignore
└── .flake8                     # Анализ кода
```

## Запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/hsemlcourse/hseml-group-project-dayaprofiii
cd hseml-group-project-dayaprofiii

# 2. Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate   # Linux/macOS
# .venv\Scripts\activate    # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запуск через расширение в VisualStudio, либо через jupyter, вводом команд в терминале:
pip install notebook
jupyter notebook
```
